Tabela do IMC


O cálculo do índice de massa corporal (IMC) é uma medida simples que relaciona o peso e a altura de uma pessoa, com o objetivo de avaliar se ela está dentro de um intervalo considerado saudável.

Para calcular o IMC, é necessário dividir o peso da pessoa pela altura ao quadrado. A fórmula matemática é a seguinte:

IMC = peso / altura²
